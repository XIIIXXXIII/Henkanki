# Feature Plan

## Available in the seed
- CLI format listing.
- CLI file inspection.
- CLI text conversions for registered text pairs.
- Web/PWA shell for local text conversion.

## Next converters
- Images: PNG, JPG, WEBP, AVIF, SVG, GIF, TIFF, ICO.
- Documents: PDF tools, Markdown, HTML, Office formats.
- Archives: ZIP, TAR, GZ, XZ, 7Z.
- Media: MP3, WAV, FLAC, OGG, MP4, WEBM, MKV.
- Developer formats: JSON, YAML, TOML, XML, CSV, OpenAPI.
