# Language Strategy

Henkanki may use many programming languages, but each language must have a clear role.

- TypeScript: Web UI, PWA, shared frontend logic.
- JavaScript: zero-dependency seed tools and CLI bootstrap.
- Rust: portable core, CLI, WASM, Tauri backend.
- C/C++: legacy and retro ports.
- Kotlin: Android.
- Swift: iOS/macOS native integrations.
- Python: prototypes and tooling.
- Go/Zig: portable utilities and optional services.
