# Branding

Henkanki comes from Japanese 変換器: converter or transformation device.

## Visual direction
- Japanese minimalism plus technical brutalism.
- Primary symbol: 変.
- Dark-first interface with a light theme later.
- Accent colors: red for transformation, cyan for technical energy.

## Voice
- Direct.
- Privacy-first.
- Local-first.
- Honest about platform support levels.
